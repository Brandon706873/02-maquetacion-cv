Los 7 Tetrominós

Un tetrominó es una figura geométrica compuesta por 4 cuadrados conectados. En Tetris, cada pieza tiene un color único y un comportamiento de rotación específico que el jugador debe dominar para limpiar el mayor número de líneas posibles.

Cómo está hecho

tetris.js // Núcleo del motor de colisiones
function collides(arena, player) {
  const [m, o] = [player.matrix, player.pos];
  for (let y = 0; y < m.length; y++) {
    for (let x = 0; x < m[y].length; x++) {
      if (m[y][x] !== 0 &&
         (arena[y + o.y] &&
          arena[y + o.y][x + o.x]) !== 0) {
        return true;
      }
    }
  }
  return false;
}

// Sistema de puntuación (Guideline oficial)
const SCORE_TABLE = {
  1: 100, // Single
  2: 300, // Double
  3: 500, // Triple
  4: 800, // Tetris! 🎉
};

// Ghost piece: proyección de caída
function getGhostY() {
  let gy = player.pos.y;
  while (!collides(arena, {...player, pos: {x: player.pos.x, y: ++gy}})) {}
  return gy - 1;
}

Historia de Tetris

84
1984
Nacimiento del Algoritmo
Alexey Pajitnov crea el prototipo en un Electronika 60 en Moscú, inspirado en el pentominó. El nombre combina "tetra" (cuatro) y su deporte favorito: tenis.
89
1989
Nintendo Game Boy
Tetris se convierte en el juego de lanzamiento de la Game Boy. Vendió más de 35 millones de copias, consolidando el juego como fenómeno global.
96
1996
Tetris Company
Pajitnov recupera los derechos de su creación y funda The Tetris Company. Por primera vez, el inventor del juego recibe regalías directas.
09
2009
The Guideline
Se establece el estándar oficial: colores canónicos, sistema de rotación SRS, 7-bag randomizer. Este juego sigue todas estas reglas.
24
2024
Esta Versión
Implementación fiel en JavaScript puro con ghost piece, hold, next queue, niveles progresivos y puntuación oficial. Construida como proyecto de programación.

Curiosidades

TETRIS EFFECT Jugar Tetris cambia la actividad cerebral. Después de sesiones largas, la gente reporta ver tetrominós en el mundo real — se llama el Efecto Tetris.
NASA RESEARCH La NASA estudió Tetris como herramienta para medir el agotamiento mental. El rendimiento en el juego correlaciona con la fatiga cognitiva del astronauta.
NP-COMPLETO En 2002, investigadores del MIT demostraron que encontrar la estrategia óptima en Tetris es un problema NP-completo: computacionalmente imposible de resolver en tiempo razonable.
